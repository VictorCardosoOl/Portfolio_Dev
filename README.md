Bem-vindo ao repositório do meu portfólio pessoal! Este projeto serve como uma vitrine interativa do meu trabalho, habilidades e jornada profissional como desenvolvedor front-end. Ele foi cuidadosamente elaborado com um forte foco em design web moderno, alta performance e uma experiência de usuário excepcional.

## 🎯 Sobre o Projeto

O objetivo deste portfólio é demonstrar não apenas os projetos que construí, mas também minha proficiência em criar interfaces ricas, responsivas e acessíveis. O projeto utiliza um ecossistema moderno para garantir a melhor performance e fluidez.

## ✨ Principais Funcionalidades

- **UI Moderna e Responsiva**: Layout que se adapta perfeitamente a dispositivos móveis, tablets e desktops.
- **Animações Fluidas**: Integração avançada com **GSAP** e **Lenis** para fornecer microinterações, animações de scroll complexas e uma experiência de navegação *premium*.
- **Arquitetura Modular**: Construído com componentes React escaláveis e altamente reutilizáveis, seguindo princípios de Clean Code.
- **Tipagem Estrita**: Código totalmente desenvolvido em **TypeScript** para garantir confiabilidade e escalabilidade.
- **Gerenciamento de Conteúdo Dinâmico**: Os dados de projetos e serviços são isolados da interface, facilitando futuras atualizações.

## 🛠️ Tecnologias Utilizadas

- **Core:** [React 19](https://react.dev/) & [TypeScript](https://www.typescriptlang.org/)
- **Build Tool:** [Vite](https://vitejs.dev/) (servidor de desenvolvimento ultrarrápido e build otimizado)
- **Estilização:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animações:** [GSAP](https://gsap.com/) & [Lenis](https://lenis.darkroom.engineering/) (Scroll Suave)
- **Ícones:** [Lucide React](https://lucide.dev/)
- **Formulários:** [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)
- **Roteamento:** [React Router v7](https://reactrouter.com/)

## 🏗️ Estrutura do Projeto

A arquitetura do projeto foi pensada para maximizar a manutenibilidade:

```text
src/
├── components/   # Blocos de construção visuais (Hero, Portfolio, Contact)
│   └── ui/       # Componentes base genéricos (Botões, Inputs, etc.)
├── config/       # Configurações globais
├── data/         # Dados estáticos (projetos, serviços)
├── hooks/        # Hooks customizados para lógica de negócio
├── layouts/      # Estruturas de página (Header, Footer estáticos)
├── lib/          # Utilitários e funções auxiliares
├── pages/        # Componentes de página (Home, About, etc.)
└── main.tsx      # Ponto de entrada da aplicação
```

## ⚙️ Como Executar Localmente

Siga os passos abaixo para rodar o projeto em sua máquina:

**1. Clone o repositório**
```bash
git clone https://github.com/VictorCardosoOl/Portfolio_Dev.git
cd Portfolio_Dev
```

**2. Instale as dependências**
Certifique-se de ter o Node.js instalado (recomendado v20+).
```bash
npm install
```

**3. Inicie o servidor de desenvolvimento**
```bash
npm run dev
```

**4. Build para produção**
```bash
npm run build
```
