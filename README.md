# 🚀 Frontend Developer Portfolio

Welcome to my personal portfolio repository! This project serves as an interactive showcase of my work, skills, and professional journey as a developer. It has been carefully crafted with a strong focus on modern web design, high performance, and an exceptional user experience.

## ✨ Key Features

- **Modern & Responsive UI**: Fully responsive layout that looks great on mobile, tablet, and desktop devices.
- **Smooth Animations**: Integrated with **GSAP** and **Lenis** to provide fluid micro-interactions, complex scroll animations, and a premium browsing experience.
- **Modular Architecture**: Built with scalable and highly reusable React components following the Single Responsibility Principle.
- **Clean Codebase**: Strictly typed with TypeScript to ensure reliability and fewer runtime errors.
- **Dynamic Content**: Data for projects and services is isolated from the UI, making the portfolio incredibly easy to update and maintain.

## 🛠️ Tech Stack

This project leverages a cutting-edge modern frontend ecosystem:

- **Core:** [React 19](https://react.dev/) & [TypeScript](https://www.typescriptlang.org/)
- **Build Tool:** [Vite](https://vitejs.dev/) (blazing fast development server and optimized production build)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [GSAP](https://gsap.com/) (GreenSock Animation Platform) & [Lenis](https://lenis.darkroom.engineering/) (Smooth Scrolling)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Form Management:** [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)
- **Routing:** [React Router v7](https://reactrouter.com/)

## 🏗️ Project Architecture

The repository is organized to maximize maintainability and scalability:

```text
src/
├── components/   # Visual building blocks (Hero, Portfolio, Contact, etc.)
│   └── ui/       # Generic and highly reusable base UI components
├── data/         # Static data arrays (projects, services, etc.)
├── hooks/        # Custom React hooks to isolate business logic
├── index.css     # Global styles and Tailwind configuration
└── main.tsx      # Application entry point
```

## ⚙️ Getting Started

Follow these steps to run the project on your local machine:

**1. Clone the repository**
```bash
git clone https://github.com/VictorCardosoOl/Portfolio_Dev.git
cd Portfolio_Dev
```

**2. Install dependencies**
Make sure you have Node.js installed (v20, v22, or higher recommended).
```bash
npm install
```

**3. Start the development server**
```bash
npm run dev
```

**4. Build for production**
```bash
npm run build
```
